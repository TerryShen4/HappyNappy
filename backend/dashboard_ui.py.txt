FILE: dashboard_ui.py
ROLE: the static look-and-feel of the website (dashboard). Imported by
       streamlit_dashboard.py.

PURPOSE
  Holds the chunks of styling and HTML so the main dashboard file stays focused on
  logic instead of markup. Pure presentation - no behavior.

WHAT'S INSIDE
  CUSTOM_CSS          - the CSS that styles the page (white background, the metric
                        cards, text colors).

  HEADER_HTML         - the page header block (the "Happy Nappy" title and the
                        "AI-Powered Power Nap Assistant" subtitle).

  BACKEND_ERROR_HTML  - the red "Cannot connect to backend" banner shown when the
                        dashboard can't reach the backend on port 8000.

NOTE
  This file only stores text; streamlit_dashboard.py is what actually displays it.
